# Elastic MDS

**Elastic MDS** is MetaFluent's elastic Market Data System - an enterprise-class platform for
distributing real-time and non-real-time content to scalable business applications through the
industry-standard APIs you already know: **JMS, JDBC, SQL, and REST**.

Elastic MDS is built on the **StreamLens** integration framework (its architecture).
